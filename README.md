# Real-World JavaScript Boilerplate
A boilerplate to help you get started

### Requirements
- `node >= v20`
- `npm >= v10`

### Backend

#### Installation
```bash
cd ./backend
npm install
```

### Frontend

#### Installation
```bash
cd ./frontend
npm install
```

### Caveats
This boilerplate may be missing things, quite intentionally. It's up to you to catch the missing file(s) and/or bugs.
